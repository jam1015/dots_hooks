# these scripts are used as hooks in my dotfiles repo

## Submodules registered by `add_submodules.bash`

- `.config/emacs` → [chemacs2](https://github.com/plexus/chemacs2)
- `.oh-my-zsh` → [ohmyzsh](https://github.com/ohmyzsh/ohmyzsh)
- `.spacemacs` → [spacemacs](https://github.com/syl20bnr/spacemacs) (develop branch)
- `.doomemacs` → [doomemacs](https://github.com/doomemacs/doomemacs)
- `.vim/pack/minpac/opt/minpac` → [minpac](https://github.com/k-takata/minpac)
- `.local/share/fonts/cmu-typewriter-dotted` → [cmu-typewriter-dotted](https://github.com/jam1015/cmu-typewriter-dotted) — CMU Typewriter Text fork with a dotted zero glyph; the kitty + fontconfig pieces it relies on live in `~/dotfiles/.config/kitty/kitty.conf` and `~/dotfiles/.config/fontconfig/fonts.conf`.
